# Pricing & Add-ons Library

Copy/paste blocks into proposals. **Edit numbers** per job; keep **one primary package** visible on page one.

---

## Core packages (choose one structure per proposal)

### A. Single package (simplest for family / private clients)

**Full production — USD $_____**

*Includes:* pre-production (outline + question map + archival list), one production day, post through approved fine cut, captions, master + streaming export.

*Does not include:* marketing assets, extra shoot days, travel outside [region], custom score, unlimited revisions.

---

### B. Tiered (good for “anchor / upsell” psychology)

**Tier A — Strategy & curation — $_____**  
Archival audit, creative brief, interview question map, optional supervisor on shoot day via remote.

**Tier B — Proof-of-concept — $_____**  
Short film (e.g.10–20 min) or assembly edit from existing footage + direction notes.

**Tier C — Full production — $_____**  
Full documentary package as in Option A.

---

### C. Phased milestones (good for grants / cautious clients)

1. **Discovery deposit — $_____ or ___%** — outline + question list + contract.  
2. **Production — $_____** — shoot day(s).  
3. **Post — $_____** — edit + one revision round + delivery.

## Add-ons (always label “quoted separately” unless included)

| Add-on | Notes | Typical trigger |
|--------|--------|-----------------|
| **Second production day** | Travel, setup, overtime | Long runtime, many subjects |
| **Extra interviews** (series) | Location changes, releases | Docu-series ep 2+ |
| **Archival scanning / restoration** | Outsourced or time-boxed | Print-only photos |
| **Custom music / composer** | License vs. work-for-hire | Broadcast / “signature” sound |
| **Extended raw footage package** | Drive handoff, labeling | Client insists |
| **Rush post** | Calendar compression | Hard premiere date |
| **Additional revision rounds** | Per round fee | Committee feedback |
| **Social / marketing suite** | Reels, trailers, thumbnails | YouTube/IG push |
| **Platform setup** | Channel art, descriptions, playlists | Ongoing presence |
| **Color / sound theatrical pass** | For festival DCP | Optional |

---

## Standard exclusions (trim to what’s true for you)

- Music, stills, or footage **not cleared** by client or rights holder.  
- **Venue / permit** fees, unless itemized.  
- **Travel & lodging** outside agreed mileage/region.  
- **Caption translation** (non-English) unless added.  
- **Insurance certificate** for institutional clients—note if COI available.

## Music boilerplate (short)

> **Score & library music** will be sourced from properly licensed libraries or original work with clear ownership. If you request recognizable commercial tracks, licensing is **client-funded** or we substitute licensed alternatives.

## Revision boilerplate (short)

> **One structured revision round** is included on the fine cut (timestamped notes). Additional rounds billed at **$___/round** or folded into a **post-production retainer**.
