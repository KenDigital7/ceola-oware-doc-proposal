# Pre-Send Checklist

Use before PDF export or email.

## Client & accuracy

- [ ] Client name / spelling correct throughout  
- [ ] Subject names (parents, orgs) **as client wants them written**  
- [ ] No transcript typos in permanent form (e.g. misheard place names)  
- [ ] Price matches what you intend: **one number**, no contradictions  
- [ ] Inclusions vs exclusions are **mirror clear** (no “gotchas”)

## Scope

- [ ] **Runtime** assumption stated or “TBD in pre-production”  
- [ ] **# shoot days** stated  
- [ ] **Deliverables** table matches body text  
- [ ] **Captions / raw / retention** explicitly addressed  
- [ ] **Travel region** or fee trigger defined if applicable  

## Tone & voice

- [ ] Sounds like **your** brand (care + craft, not desperate)  
- [ ] At least **two phrases** echo client’s language from call  
- [ ] No industry jargon without plain-English gloss  

## Risk & ethics

- [ ] Sensitive topics handled **without exploiting** trauma for drama  
- [ ] Elders / health: **comfort, pacing, breaks** implied or stated  
- [ ] If diaspora / culture: **no cartoon framing**—specific, respectful  

## Legal-lite (non-lawyer sanity)

- [ ] “Subject to agreement” or link to **your contract** if you use one  
- [ ] Music / archival clearance responsibility stated  
- [ ] Revision count stated  

## Next step

- [ ] **One clear CTA** (reply with dates / approve scope / schedule call)  
- [ ] Contact info correct  

## File hygiene

- [ ] Remove internal comments / bracket notes like `[your email]`  
- [ ] Filename: `Proposal-[ClientSlug]-[YYYY-MM].pdf`  

## Optional power move

- [ ] Attach **one-pager archival checklist** (reduces client overwhelm)  
- [ ] Offer **15-min tech scout** (photo FaceTime of rooms) if in-home
